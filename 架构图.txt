架构：
    MD任务生成器
    MD任务执行器

任务触发 -> 判断主站状态 -> MD任务执行 -> 更新dom
                        -> 等待主站空闲 -> MD任务执行 -> 更新dom
modbus处理：
    更新主站状态忙碌 -> 生成指令 -> 发送指令 -> 接收指令 -> 解析指令 -> 更新主站状态空闲 -> 更新dom

modbus指令发送流程：
    1. 指令执行
    2. 生成指令
    3. 发送指令
    4. 读取响应（读器）
    5. 解析数据（读器）
    6. 获得数据
    7. 格式化数据返回
